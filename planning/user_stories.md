# Outline User Roles

## Member

A user who wants to build consistent fitness habits through personalized workout plans, progress tracking, and accountability groups.

## Accountability Group Leader

A member who creates and manages an accountability group, inviting others to stay motivated and consistent together.

> **Note:** A member can also be an Accountability Group Leader.

# Draft User Stories

1. As a **Member**, I want to complete a short onboarding questionnaire, so that the app can generate workout plans tailored to my fitness goals, experience level, available equipment, and workout schedule.

2. As a **Member**, I want to see my personalized daily workout immediately after opening the app, so that I don't have to spend time deciding what exercises to do.

3. As a **Member**, I want to view detailed instructions for today's workout in a slide-out panel or modal, so that I can understand each exercise without leaving the dashboard.

4. As a **Member**, I want to mark my daily workout as completed, so that my progress, streak, and workout history are automatically updated.

5. As a **Member**, I want to view my workout history, consistency streaks, and achievement milestones, so that I can measure my long-term progress and stay motivated.

6. As a **Member**, I want to edit or delete a workout log if I accidentally recorded incorrect information, so that my workout history remains accurate.

7. As a **Member**, I want to browse and filter the exercise library by muscle group, equipment, or difficulty level, so that I can learn more about the exercises included in my workout plans.

8. As an **Accountability Group Leader**, I want to create an accountability group, so that I can invite friends or workout partners to stay consistent together.

9. As a **Member**, I want to join an accountability group, so that I can stay motivated by working toward my fitness goals alongside others.

10. As an **Accountability Group Leader**, I want to view each group member's workout completion status and current streak, so that I can encourage members and celebrate their progress.

11. As a **Member**, I want the app to automatically generate a personalized starter workout plan after I complete onboarding, so that I can begin my fitness journey immediately without creating my own workout routine.

12. As a **Member**, I want to receive personalized motivational messages based on my recent progress and consistency, so that I feel encouraged to continue building healthy habits.
